# Neuroscience Dataset Assembly

## Approach

1. **Scraped BioRxiv** - Collected all neuroscience papers from BioRxiv
2. **Direct citation analysis** - Found 0 direct citations between BioRxiv neuroscience papers and neuromorphic papers in our dataset
3. **Common citations** - Identified ~200 papers cited by both neuroscience and neuromorphic groups
4. **Two-hop expansion** - Expand two hops out from those papers into the BiorXiv dataset
